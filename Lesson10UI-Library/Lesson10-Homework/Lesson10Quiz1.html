Answer:

  1. Describe in your own words how the "master" slider is created in the example code:
    • On page initialization, the jquery slider method: (1) loads quite a bit of styling and (2) adds an anonymous function to the "master" div # as shown here:
    • There is a span element added as well by jquery that handles the state of the slider such as where it is on the screen, how many pixels from the left it has been moved, if the handle is hovered on, clicked on/active, etc (you can see that in above snapshot as well).
  2. what method is used to create the slider widget?
    • slider();
  3. What do you think the options that are passed into this method do?
    • value: 60,

    • orientation: "horizontal",
      • Sets the slider handles to be horizontal
    • range: "min",
      • Displays from the minimum value to the handle
    • animate: true
      • Defines that the slide handle animates for default duration

Question 4:

Describe in your own words the differences between a widget and an interaction in the jQuery UI library.

Answer:

  • Widget:
    • A Widget is an actual control of some sort like an accordion, button, menu, dialog box, etc.
  • Interaction:
    • An interaction adds 'behavior' to the widgets such as: 'draggable', 'resizeable', 'sortable', etc.
