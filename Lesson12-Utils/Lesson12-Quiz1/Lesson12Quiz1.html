Question 2:

Look at the code below:

var a1 = [1, 3, 5];
var a2 = [2, 4, 6];
$.merge(a1, a2); 
alert(a1);

What is displayed in the alert? Why?

Answer 2:

1,3,5,2,4,6

the .merge method takes the second array and merges it with the first, maintaining the order of the elements.

Question 3

Describe the difference between a jQuery method, like addClass(), and a jQuery utility function, like $.trim().

Answer 3:

  • jQuery methods, like addClass(), are functions bound to various html elements in the DOM, usually implemented with a statement like:
    $("div#hello").addClass('myClass');
  • jQuery utilities, on the other hand, like $.trim(), provide you the ability to accomplish routine tasks such as removing whitespace, merging arrays or finding the current time. These utilities operate within the $ namespace rather than being bound to specific DOM elements.
