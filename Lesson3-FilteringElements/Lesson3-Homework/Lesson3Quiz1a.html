Question 1:
In the lesson, we wrote a function, filterChecked(), to add a highlight to all checked items. We used the parent() method to get 
the li element that is the parent of the matching input element, to highlight (since if we highlight the checkbox, we won't see much).

Suppose we modify the code like this:

var filterChecked = function() {
    $("li input:checked").parent()next().addClass("highlight");
};    
Describe which element is highlighted by the filterChecked() function now. Use the documentation at jQuery.com to look up the 
next() method and describe what it does and how we're using it in this function.
With the next() function, you navigate to the next sibling of each element in the set of matched elements in the DOM.
