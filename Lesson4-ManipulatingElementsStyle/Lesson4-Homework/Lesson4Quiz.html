Question 1:
Look at the following jQuery:

$("div").width(300);
console.log("The width of the div is: " + $("div").width());
console.log("The width of the div is: " + $("div").css("width"));    
What two values will you see in the alert for the width? Explain the difference.
Answer: 
The width of the div is: 300
The width of the div is: 300px 

quoted from jquery documentation:

The difference between .css(width) and .width() is that the latter returns a unit-less pixel value (for example, 400) while the former 
returns a value with units intact (for example, 400px). The .width() method is recommended when an element's 
width needs to be used in a mathematical calculation.

Question 2:

If you wanted to get the width of an element, including the padding, the border, and the margin, what jQuery 
method would you call?
Answer: 
Use the .outerWidth() method, set the boolean parameter to 'true' to add the margin value:

console.log("The width of the div with padding and border and margin is: " + $("div").outerWidth( true ));

question 3:
What does the toggleClass() method do?


Answer:


In the simplest version, if an element in the matched set of elements already has the class, then it is removed; 
if an element does not have the class, then it is added.
