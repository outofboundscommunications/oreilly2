Question 1:
Describe in your own words what each() does.
Answer: 
the each() function is a way to loop thru each element of a jquery object of elements you
have selected.

it is usually used when you want to make more complicated changes to elements selected by a selector than just changing their color or highlighting them.

each() is the preferred way to loop in JQuery and it compares to the standard javascript 'for' loop.
